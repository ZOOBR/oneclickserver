В данном наборе есть все необходимое для запуска игр, для установки скаченных NusGrabber-ом апдейтов и для доступа в магазин без обновления.

Установка.

1. Из папки vc_redist запускаем и устанавливаем vc_redist.x86.exe.
2. Если у вас уже стоит http-сервер на компьютере, то блокнотом открываем файл ServerPort.txt и меняем порт 80 на любой другой, например 8630.
2. Запускаем start.cmd
3. Из браузера консоли открываем адрес своего компьютера по IP или имени. Если вы во втором пункте указали свой порт, то его нужно указать через : после адреса

Если у вас стоит TubeHax, то используем только по IP, то есть например http://192.168.1.8/, по имени работать с внешним DNS не будет.

Если имя компьютера user-pc, то адрес будет такой:
Со стандартным портом:
http://user-pc/
Со своим портом:
http://user-pc:8630/
Если так не заработает значит, заменяем имя компьютера его IP-адресом.

Если вы хотите поставить сервер с хаками как службу, чтобы она запускалась при старте компьютера незаметно для вас то вам просто необходимо запустить файл install.cmd, соответственно для удаления службы запускаете uninstall.cmd. Для установки на компьютере должен быть NET Framework 4.5.2 или выше. Просто для запуска он не нужен.

После запуска на страничке сервера, вы увидите 3 ссылки:
Loadiine GX 0.2 - для запуска игр в формате loadiine
wupInstaller - для загрузки обновлений на игры
nnu patcher - для пользования магазином с включенным TubeHax
