# Однокнопочный сервер с эксплоитами для Wii U

В данном наборе есть все необходимое для запуска игр, для установки скаченных NusGrabber-ом апдейтов и для доступа в магазин без обновления.

# Подготовительные работы )
Следующие шаги нужны тем, кто не запускал вообще раньше игры для Loadiine, то ессть новичкам только что купившим консоль и жаждущим запускать скаченные игры. Остальные идут сразу к разделу "Запуск/установка сервера"

1. Выдергиваем из роутера кабель провайдера интернета;
2. Включаем консоль и убеждаемся что версия прошивки 5.3.2 или 5.4.0, на других работать не будет, если прошивка ниже надо обновить ее с лицензионного диска (перечень игр с нужной прошивкой ограничен, например подойдет диск Splatoon);
2. Настраиваем интернет на консоли,IP-адрес: Авто, DNS: Вручную, Предпочтительный DNS: 107.211.140.065 (TubeHax DNS) Альтернативный DNS: 0.0.0.0;
3. Форматируем карту памяти в SD с размером кластера 32 или 64 Кб, лучше второе;
4. Закидываем папку wiiu из скаченного архива на вашу карту с перезаписью;
5. Закидываем игры, подготовленные для Loadiine в папку games внутри папки wiiu, папка с игрой должна содержать папки code,content и опционально паку meta. Называться папка с игрой должна по следующему шаблону <Название игры>[Код игры], например Little Inferno [WLFPXX];
6. Итак, убеждаемся, что все сделали правильно и подключаем кабель провайдера к роутеру;

# Запуск/установка сервера с эксплоитами на локальном сервере

2. Если у вас не установлен набор библиотек Visual C++ Redistributable for Visual Studio 2015, то из папки vc_redist запускаем и устанавливаем vc_redist.x86.exe.
3. Если у вас уже стоит http-сервер на компьютере, то блокнотом открываем файл ServerPort.txt и меняем порт 80 на любой другой, например 8630.
4. Запускаем start.cmd
5. Из браузера консоли открываем адрес своего компьютера по IP или имени. Если вы во втором пункте указали свой порт, то его нужно указать через : после адреса
Если у вас прописан TubeHax DNS в настройках консоли, то используем только IP, то есть например http://192.168.1.8/, по имени работать с внешним DNS не будет.
Если имя компьютера user-pc, то адрес будет такой:
Со стандартным портом:
http://user-pc/
Со своим портом:
http://user-pc:8630/
Если так не заработает значит, заменяем имя компьютера его IP-адресом.

Если вы хотите поставить сервер с хаками как службу, чтобы она запускалась при старте компьютера незаметно для вас то вам просто необходимо запустить файл install.cmd от имени Администратора (по правой кнопке мыши), соответственно для удаления службы запускаете uninstall.cmd (также от Администратора). Для установки на компьютере должен быть NET Framework 4.5.2 или выше. Просто для запуска он не нужен.

После запуска на страничке сервера, вы увидите несколько ссылок:

Loadiine GX2 v0.2 - для запуска игр подготовленных для загрузчика loadiine GX2. Это новая версия, не требующая перезапуска браузера. Старую версию приходилось два раза запускать.
Homebrew Launcher v1.1 - Лончер elf - файлов, то есть загрузчик программ в формате elf, в том числе Loadiine
wupinstaller - для установки обновлений для игр
nnu patcher - для пользования магазином с включенным TubeHax
