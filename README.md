# Однокнопочный сервер с эксплоитами для Wii U под Андроид

В данном наборе есть все необходимое для запуска игр, для установки скаченных NusGrabber-ом апдейтов и для доступа в магазин без обновления.
Конечно, в отличии от набора для Windows этот пак назвать совсем однокнопочным нельзя)), но создавать под это дело отдельный репозиторий я не стал. 

# Подготовительные работы )
Следующие шаги нужны тем, кто не запускал вообще раньше игры для Loadiine, то ессть новичкам только что купившим консоль и жаждущим запускать скаченные игры. Остальные идут сразу к разделу "Запуск/установка сервера"

1. Убеждаемся что на вашем смартфоне отключен интернет;
2. Включаем консоль и убеждаемся что версия прошивки 5.3.2 или 5.4.0, на других работать не будет, если прошивка ниже надо обновить ее с лицензионного диска (перечень игр с нужной прошивкой ограничен, например подойдет диск Splatoon);
3. Включаем режим Wi-Fi точки доступа на вашем устройстве (на разных устройствах это может делаться по разному)
2. Настраиваем интернет на консоли на вашу новоиспеченную точку доступа,IP-адрес: Авто, DNS: Вручную, Предпочтительный DNS: 107.211.140.065 (TubeHax DNS) Альтернативный DNS: 0.0.0.0;
3. Форматируем карту памяти в SD с размером кластера 32 или 64 Кб, лучше второе;
4. Закидываем папку wiiu из скаченного архива на вашу карту с перезаписью;
5. Закидываем игры, подготовленные для Loadiine в папку games внутри папки wiiu, папка с игрой должна содержать папки code,content и опционально паку meta. Называться папка с игрой должна по следующему шаблону <Название игры>[Код игры], например Little Inferno [WLFPXX];

# Запуск/установка сервера с эксплоитами на смартфоне или планшете c ОС Android

1. Закидываем папку www из архива на ваш девайс, можно как на внешнюю SD так и на внутреннюю память;
2. В магазине приложений находим и устанавливаем программу "Сервер для PHP";
3. При первом запуске программы будет предложено выбрать сборку PHP, если вам это ни о чем не говорит, просто выбираем последнюю версию;
4. Выбираем в качестве основного каталога папку, которую закинули в пункте 1;
5. Выбираем необходимый порт сервера (по умолчанию он 8080, можно таким и оставить) и нажимаем кнопку "ЗАПУСК СЕРВЕРА";
6. Из браузера консоли открываем адрес своего сервера по IP, например http://192.168.1.8:8080 (если что, это адрес вы найдете в запущенной программе).

После запуска на страничке сервера, вы увидите несколько ссылок:

Loadiine GX2 v0.2 - для запуска игр подготовленных для загрузчика loadiine GX2. Это новая версия, не требующая перезапуска браузера. Старую версию приходилось два раза запускать.
Homebrew Launcher v1.1 - Лончер elf - файлов, то есть загрузчик программ в формате elf, в том числе Loadiine
wupinstaller - для установки обновлений для игр
nnu patcher - для пользования магазином с включенным TubeHax
